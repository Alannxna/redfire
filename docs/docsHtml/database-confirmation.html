🏗️ 数据库架构中的角色

数据类型 存储位置 说明
用户数据 MySQL 用户信息、账户、权限管理
交易数据 MySQL 订单、成交记录、持仓信息
策略数据 MySQL 策略配置、回测结果、参数设置
系统配置 MySQL 系统配置、日志记录、监控数据
实时缓存 Redis 会话、临时数据、消息队列
时序数据 InfluxDB 市场行情、K线数据、指标数据
